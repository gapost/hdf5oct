HDF5 file I/O

h5create Create a HDF5 dataset.
h5write Write data to a HDF5 dataset.
h5writeatt Write a HDF5 attribute.
h5read Read data from a HDF5 dataset.
h5readatt Read a HDF5 attribute.
h5info Return information about the content of a HDF5 file.
h5disp Display the contents of a HDF5 file.
h5load Load an entire HDF5 file or a portion of it as a struct.
